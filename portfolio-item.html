Fortune Cookie

Learn your fate

The purpose of this project was to learn how to manipulate the DOM using JavaScript and learn how to use an API. The DOM tree is edited every time the cookie is clicked on, resulting in a different phase of cookie. To display the fortune, a call is made to the AdviceSlip json API and the DOM is edited again to show the text.

You can read more on the project and see the code on my github
